## Code Examples

### Inline Code
Use the `useState` hook to manage state. The `useEffect` hook handles side effects. Import with `import React from 'react'`.


```ts title="typescript"
const array = ["apple", "banana", "orange"];

const object = array.reduce((acc, value, index) => {
	acc[`fruit ${index + 1}`] = value;
	return acc;
}, {} as { [key: string]: string });

console.log(object);

// Output: { 'fruit 1': 'apple', 'fruit 2': 'banana', 'fruit 3': 'orange' }
```
