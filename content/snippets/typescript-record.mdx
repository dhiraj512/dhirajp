The Record utility type allows you to create an object type whose property keys are of a specific type, and a consistent value type.

```ts title="Record.ts"
type Fruit = "apple" | "banana" | "orange";
//Fruit type can be either "apple", "banana", or "orange"
type FruitInventory = Record<Fruit, number>;
//FruitInventory type is an object with Fruit keys and number values
const fruitInventory: FruitInventory = {
	apple: 10,
	banana: 20,
	orange: 30,
};
console.log(fruitInventory.apple); //10
console.log(fruitInventory.banana); //20
console.log(fruitInventory.orange); //30
```
