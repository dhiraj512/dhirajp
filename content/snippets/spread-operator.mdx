The spread operator is a useful and quick syntax for adding items to arrays,
combining arrays or objects, and spreading an array out into a function’s arguments.

It's used for a few different purposes depending on the context.

## Copying an array

```js
const arr = [1, 2, 3];
const newArr = [...arr];
// Output: [1, 2, 3]
```

## Concatenating or combining arrays

```js
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const newArr = [...arr1, ...arr2];
// Output: [1, 2, 3, 4, 5, 6]
```

## Adding items to an array

```js
const arr = [1, 2, 3];
const newArr = [...arr, 4];
// Output: [1, 2, 3, 4]
```

## Spreading an array into an object

```js
const arr = [1, 2, 3];
const obj = { ...arr };
// Output: { 0: 1, 1: 2, 2: 3 }
```

## Spreading an array into a function’s arguments

```js
const arr = [1, 2, 3];
const sum = (a, b, c) => a + b + c;
sum(...arr);
// Output: 6
```

## capturing the rest of an array

```js
const arr = [1, 2, 3, 4, 5];
const [first, second, ...rest] = arr;
// Output: first = 1, second = 2, rest = [3, 4, 5]
```

## Desctructuring an object

```js
const person = { firstName: "John", lastName: "Doe", age: 30 };
const { firstName, ...rest } = person;
// Output: firstName = 'John', rest = { lastName: 'Doe', age: 30 }
```
