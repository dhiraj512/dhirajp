### HTML element

```html
<div class="centerthis">Hello world!</div>
```

### Using CSS

```css
html,
body {
	height: 100vh;
}
.centerthis {
	display: flex;
	align-items: center;
	justify-content: center;
}
```

### Using TailwindCSS

```css
<div class="h-screen flex justify-center items-center">
  <div>Centered</div>
</div>
```
